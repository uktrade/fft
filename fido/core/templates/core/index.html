Welcome to FIDO

(Financial Information Delivered Online)

This is an in-house bespoke web application supporting business critical processes in Finance. This will replace the admin consolidation and analysis tool (‘Admin Tool’), currently used for the financial management of administrative resources and staff forecasts once all functionalities have transferred over.

FIDO has been created to;

  • assist you in managing your financial resources, and
  • streamline current manual processes.

If you require access or need to report issues, please email FIDO@trade.gov.uk
