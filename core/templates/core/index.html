With FFT you can:

  • manage financial forecasts
  • manage payroll forecasts
  • view the Universal Chart of Account for DBT
  • record gifts and hospitality given to civil servants

Changes to the tool

The Financial Forecasting Tool (FFT) replaces Admin Tool as of 17th April. FFT covers all forecasting processes, for:

  • programme, capital and AME (annually managed expenditure) - previously done in spreadsheets
  • payroll and admin - previously done in Admin Tool

Deadlines for UK SBS - Oracle

Payroll
