Financial Forecasting Tool

The Financial Forecasting Tool replaces the admin consolidation and analysis tool ("Admin Tool"), currently used for the financial management of resources and staff forecasts.

This is an in-house bespoke web application supporting business critical processes within the DBT.

The Financial Forecasting Tool supports three functions:

  • Viewing the DBT chart of account
  • Registering gifts and hospitality
  • Managing financial forecasts: assists you in managing your financial resources
