The Financial Forecasting Tool (FFT) replaces Admin Tool.

Forecasting used to be done using two separate processes. Spreadsheets were used for programme, capital and AME (Annually Managed Expenditure). Payroll and Admin were then done separately in Admin Tool.

With FFT you can;

  • manage financial forecasts
  • manage payroll forecasts
  • view the Chart of Account for DBT
  • record gifts and hospitality given to civil servants
