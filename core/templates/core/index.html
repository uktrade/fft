Financial Forecast Tool

This is an in-house bespoke web application supporting business critical processes within the DIT.

The Financial Forecast Tool supports three functions:

  • Viewing the DIT chart of account
  • Registering gifts and hospitality (disabled for now)
  • Managing financial forecasts

Managing financial forecasts

The Financial Forecast Tool forecast function replaces the admin consolidation and analysis tool ("Admin Tool"), currently used for the financial management of administrative resources and staff forecasts.

The Financial Forecast Tool forecast function has been created to;

  • assist you in managing your financial resources, and
  • streamline current manual processes.

If you require access or need to report issues, please email fft@trade.gov.uk
